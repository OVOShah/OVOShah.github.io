    Gainsbot
    Gainsbot is a joystick controlled arm that can intelligently remember its movements. The arm is controlled by an Arduino UNO, which recieves signals from two joysticks and communicates them to the 4 servos. With the push of a button, gainsbot will track its movements and at the end of a fixed time interval, replay those movements continously.

    Arduino code and CAD files can be found on GitHub here.

    Jeepbot
    Jeepbot is the Android controlled car of the future. The heart of the car is an Arduino Uno, which parses signals recieved by the bluetooth module and configures the power and polarity delivered to each motor to move the car in the desired direction. An ultrasonic sensor shuts off motors and automatically backs up the vehicle if it is too close to any object, preventing collisions. All of the car's abilities including the lights can be toggled from the Android app. Both Arduino code and Android app have been designed by me, and can be found on GitHub here.
